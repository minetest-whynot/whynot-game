# Orbs of Time

A mod containing magical orbs a player can use a limited number of times to change the time of day.
by: Philipbenr And DanDuncombe

Licence: [MIT](LICENSE)

![screenshot](screenshot.png)

Orbs present in this mod are:

 * Orb of Midday
 * Orb of Dusk
 * Orb of Midnight
 * Orb of Dawn
