# Sound API

mod that enables sound to be game agnostic
## Installing

* `git clone https://github.com/mt-mods/sound_api.git`
* `cd sound_api`
* `git submodule init`
* `git submodule update`

to update please use the following commands starting inside the mod directory

* `git pull`
* `git submodule sync`
* `git submodule update`
