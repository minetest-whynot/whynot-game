# Mobs Ghost Redo

Originally created by Hamlet <hamlatmesehub@riseup.net> and contributors.

Refactored by TenPlus1.

**NOTE:** This mod uses Blockmen's [CME Ghost][8] mob ported to Mobs Redo API.

**Dependencies:** mobs ([Mobs Redo][4])

**Supported:** default, bones (found in [Minetest Game][3]), [mobs_humans][5], [mobs_others][6], [moreores][7]

__Advanced options:__
Settings -> All Settings -> Mods -> mobs_ghost_redo

'mobs_ghost_redo_daytime_check' Set to True to kill ghosts during daytime only.

'mobs_ghost_redo_bones_only'    Set to True so that ghosts only spawn on bones.

'mobs_ghost_redo_difficulty'    Set to True so that only specific weapons harm ghosts.
